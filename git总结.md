# 本节课的指令总结
1. ls 查看
1. cd 进入路径
1. mkdir 创建新文件夹
1. touch 创建新文件
1. rm -rf 删除文件（不可逆）
1. cd .. 返回上级菜单
1. sudo docker stop xxx停止xxx
1. sudo docker rm xxx删除xxx
## git类
1. ssh-keygen -t rsa -b 4096 -C "your_email@example.com" 配置ssh
1. cat ~/.ssh/id_rsa.pub 查看ssh
1. git config 全局配置
1. git clone 下载
1. git add . 修改代码  git add -A
1. git commit -m "备注" 提交代码
1. git pull 更新代码
1. git push 推送代码
