# 本节课的指令总结

1. ls 查看

2. cd 进入路径

3. mkdir 创建新文件夹

4. touch 创建新文件

5. rm -rf 删除文件（不可逆）

6. cd .. 返回上级菜单

7. sudo docker stop xxx停止xxx

8. sudo docker rm xxx删除xxx
   
   ## git类

9. ssh-keygen -t rsa -b 4096 -C "your_email@example.com" 配置ssh

10. cat ~/.ssh/id_rsa.pub 查看ssh

11. git config 全局配置

12. git clone 下载

13. git add . 修改代码  git add -A

14. git commit -m "备注" 提交代码

15. git pull 更新代码

16. git push 推送代码
